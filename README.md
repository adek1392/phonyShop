# E-Commerce Demo

A modern e-commerce demo website built with **React**, **Tailwind CSS**, and **React Router**, fetching products from [FakeStoreAPI](https://fakestoreapi.com/).  

---

## Live Demo

[View the live demo on Vercel](https://phony-shop-jxq13es9g-adrians-projects-06f1dc67.vercel.app/)

## Features

- Responsive product catalog with categories  
- Product details and checkout confirmation page  
- Newsletter subscription with custom validation  
- Smooth client-side navigation  

---

## Tech Stack

- **React** – UI library  
- **React Router** – Routing  
- **Tailwind CSS** – Styling  
- **FakeStoreAPI** – Products data
